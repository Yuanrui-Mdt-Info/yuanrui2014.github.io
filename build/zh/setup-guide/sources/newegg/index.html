设置指南​

第一步：获取Newegg设置相关信息​

  1. 登录您的Newegg账号。

  3. 您已成功获取了所有在Daspire设置Newegg所需的资料。

第二步：在Daspire中设置数据源​

  1. 在数据源设置页面上，从数据源类型下拉列表中选择Newegg。

  2. 输入数据源名称。

  3. 输入Newegg

  4. 输入Newegg

  5. 在数据复制方式中，在根据开始日期复制或周期性复制中选择。

  6. 点击保存并测试。

支持的数据流​

此数据源能够同步以下数据流：

获取物品库存（Get Item Inventory） 获取批量库存（Get Batch Inventory） 获取商品价格（Get Item Price） 获取批量价格（Get Batch Price） 获取订单信息（Get Order Information） 获取额外订单信息（Get Additional Order Information） 获取RMA信息（Get RMA Information） 获取礼遇退款信息（Get Courtesy Refund Information） 获取礼遇退款请求状态（Get Courtesy Refund Request Status） 获取报告状态（Get Report Status） 获取订单列表报告（Get Order List Report） 获取结算摘要报告（Get Settlement Summary Report） 获取结算交易报告（Get Settlement Transaction Report） 获取RMA清单报告（Get RMA List Report） 获取商品基本信息报告（Get Item Basic Information Report） 获取商品查询报告（Get Item Lookup Report） 获取每日库存报告（Get Daily Inventory Report） 获取每日价格报告（Get Daily Price Report） 活动列表（Campaign List） 组列表（Group List） 活动绩效报告（Campaign Performance Report） 获取入站货件状态请求（Get Inbound Shipment Status Request） 获取入库货件请求结果（Get Inbound Shipment Request Result） 获取入库货件列表（Get Inbound Shipment List） 获取仓库列表（Get Warehouse List）

故障排除​

单次可同步的最大表数为6千张。如果由于达到最大表数而无法获取数据架构，我们建议您调整数据源设置。
