Amazon Selling Partner

This page contains the setup guide and reference information for the Amazon Selling Partner (SP).

Prerequisites

  1. app_id

  2. lwa_app_id

  3. lwa_client_secret

  4. refresh_token

  5. aws_access_key

  6. aws_secret_key

  7. role_arn

  8. aws_environment

  9. region

  10. replication_start_date

Setup guide

Step 1: Set up Amazon Selling Partner

Register Amazon Seller Partner application. Create IAM user.

Step 2: Set up the Amazon Selling Partner data source

  1. Select Amazon Selling Partner from the Source list.

  2. Using developer application from Step 1, generate refresh token.

  3. Paste all data to required fields using your IAM user and developer account.

  4. Click Save & Test.

Supported sync modes

The Amazon SP data source supports the following sync modes:

  1. Full Refresh

  2. Incremental

Supported streams

This source is capable of syncing the following streams:

  1. FBA Inventory Reports

  2. FBA Orders Reports

  3. FBA Shipments Reports

  4. FBA Replacements Reports

  5. FBA Storage Fees Report

  6. Restock Inventory Reports

  7. Flat File Open Listings Reports

  8. Flat File Orders Reports

  9. Flat File Orders Reports By Last Update (incremental)

  10. Amazon-Fulfilled Shipments Report

  11. Merchant Listings Reports

  12. Vendor Direct Fulfillment Shipping

  13. Vendor Inventory Health Reports

  14. Orders (incremental)

  15. Seller Feedback Report (incremental)

  16. Brand Analytics Alternate Purchase Report

  17. Brand Analytics Item Comparison Report

  18. Brand Analytics Market Basket Report

  19. Brand Analytics Repeat Purchase Report

  20. Brand Analytics Search Terms Report

  21. Browse tree report

  22. Financial Event Groups

  23. Financial Events

Report options

Make sure to configure the required parameters in the report options setting for the reports configured.
