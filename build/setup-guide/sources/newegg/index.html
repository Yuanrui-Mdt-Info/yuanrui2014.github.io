Setup guide​

Step 2: Set up Newegg in Daspire​

  1. Select Newegg from the Source list.

  2. Enter a Source Name.

  3. Enter your Newegg

  4. Enter your Newegg

  5. In Data Replication Schedule, choose an option between Based on Start Date or Periodic Replication.

  6. Click Save & Test.

Supported streams​

This source is capable of syncing the following streams:

Get Item Inventory Get Batch Inventory Get Item Price Get Batch Price Get Order Information Get Additional Order Information Get RMA Information Get Courtesy Refund Information Get Courtesy Refund Request Status Get Report Status Get Order List Report Get Settlement Summary Report Get Settlement Transaction Report Get RMA List Report Get Item Basic Information Report Get Item Lookup Report Get Daily Inventory Report Get Daily Price Report Campaign List Group List Campaign Performance Report Get Inbound Shipment Status Request Get Inbound Shipment Request Result Get Inbound Shipment List Get Warehouse List

Troubleshooting​

Max number of tables that can be synced at a time is 6,000. We advise you to adjust your settings if it fails to fetch schema due to max number of tables reached.
